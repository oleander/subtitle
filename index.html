Subtitle

A drag 'n drop application for opensubtitles.org

Download for Windows Download for Linux Download for OS X

What does it do?

Subtitle let's you automatically download subtitles for you current media file using just drag 'n drop. It looks at the video name and check with opensubtitles.org if the subtitle is avalible. If it is, it will download the subtitle in the background, unpack it and place it in the same folder as the original video.

How do I use it?

Drag the your movie or tv-show into the drop area. Wait for the okay sign and head back to the original folder where the media was located. The downloaded subtitle should now be visible in the selected language.

How do i install it?

Download the application for your operating system above. Unzip the folder in the applications folder and click on the application icon. You're done!

How do I change language?

Press on the big, blue button to change language.
